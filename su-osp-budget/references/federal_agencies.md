## Special Cases

### Federal Pass-Through Awards
When a non-federal entity (state, foundation, etc.) distributes federal funds:
- Check if award references 2 CFR 200 (Uniform Guidance)
- Check for Federal Award Identification Number (FAIN)
- If yes → treat as **federal** for fringe rate purposes

### State Government Using Federal Funds
- Ask user to verify if federal cost principles apply
- If award terms reference 2 CFR 200 → federal rates apply

## Non-Federal Sponsors

When sponsor is NOT in the above list, assume non-federal unless:
- Award terms explicitly reference federal regulations
- User confirms it's a federal pass-through

**Common Non-Federal Sponsors:**
- Private foundations (Gates, Ford, Mellon, Sloan, Moore, etc.)
- Non-profit organizations (American Heart Association, ACS, etc.)
- Corporations and industry
- State and local governments (when using own funds)
- Foreign governments
- Professional associations

## Usage in Skill

When user mentions a sponsor name:
1. Check this list to determine federal vs. non-federal
2. If found in federal list → use federal fringe rates
3. If not found → ask for clarification or assume non-federal (state assumption)
4. For graduate assistants, federal vs. non-federal matters significantly (10.8% vs. 55.0%)
